FreshCart

It includes an admin dashboard for managing products and orders, and is built with a scalable tech stack optimized for performance and mobile use.

Ecommerce Watch Store

The eCommerce Watch Store is an online platform built with the MERN stack (MongoDB, Express.js, React.js, Node.js), allowing users to browse and purchase watches securely. It features product listings, cart functionality, and a checkout process with JWT-based user authentication. The site is responsive, providing an optimized shopping experience across devices.

CineScope

Cinescope is a movie rating and review platform where users can explore and manage their favorite movies. Built with the MERN stack, Cinescope allows users to add movies to their personal favorite list, search for movies, and leave reviews. The platform utilizes the TMDB API for movie data and integrates JWT authentication for secure user login and interactions. Cinescope offers an engaging experience for movie enthusiasts to discover, track, and share their movie preferences.
